# Certificates Folder

Yahan aap apne certificate images aur PDFs add kar sakte ho.

## Instructions (Hindi/Hinglish)

1. **Certificate Images Add Karo:**
   - Apne certificate ki image ko is folder mein copy karo
   - Image ka naam yeh hona chahiye:
     - `generative-ai-certificate.jpg` (ya .png)
     - `discrete-math-certificate.jpg`
     - `math-2-3-certificate.jpg`
     - `internship-certificate.jpg`

2. **PDF Files Add Karo (Optional):**
   - Agar aap chahte ho ki log certificate download kar sake, to PDF files bhi add kar sakte ho
   - PDF ka naam same rakho, bas extension `.pdf` hona chahiye
     - `generative-ai-certificate.pdf`
     - `discrete-math-certificate.pdf`
     - etc.

3. **File Formats:**
   - Images: JPG, PNG, WebP (recommended: JPG)
   - PDFs: PDF format

4. **File Size:**
   - Images: 1-2 MB se zyada nahi hona chahiye (better performance ke liye)
   - PDFs: 5 MB tak theek hai

## Example Structure

```
public/
  certificates/
    generative-ai-certificate.jpg
    generative-ai-certificate.pdf
    discrete-math-certificate.jpg
    discrete-math-certificate.pdf
    math-2-3-certificate.jpg
    math-2-3-certificate.pdf
    internship-certificate.jpg
    Internship-Certificate-certificate.pdf
```

## Note

Agar image nahi milti hai, to automatically placeholder image show hogi. 
Aap baad mein bhi images add kar sakte ho - bas file ko public/certificates folder mein copy karo.
